# AutoClipper App

AutoClipper is a bot that automatically finds, downloads and uploads viral clips to TikTok.

## Features

- Automatic clip detection
- Video formatting for TikTok
- Automatic upload using TikTok API
- Multiple account support

## How it works

1. Scan streamers
2. Download clips
3. Edit clips
4. Upload videos automatically

## Technologies

- Python
- TikTok Content Posting API
- Twitch API
- FFmpeg

## Permissions used

- user.info.basic
- video.upload
- video.publish

## Legal

Users must comply with TikTok terms of service.
